# Goal

검색 첫 화면
 - 최근 검색어 10개 보여주기

검색 중 화면
 - 히스토리에서 검색해서 보여주기

검색 결과 화면
 - 검색 결과 화면은 앱스토어와 동일하게 보여주기

상세 화면
 - 설명, 이미지

설계
 - 스토리보드 오토레이아웃
 - 자체 네크워크 개발
 - 이미지 다운로드 캐싱 개발

기타
 - 팟 포함 되어 있음
